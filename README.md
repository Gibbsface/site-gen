
# Under the Hood
The parser will read `markdown.md` in the root of this directory. It can only handle the following block types and text decorators:

- Blocks
- - Code (multiline)
- - Paragraph
- - Image
- - Heading (all sizes)

- Text Decorations
- - Code (inline)
- - Link
- - Bold
- - Italic

# Instructions
run with `./main.sh`
test with `./test.sh`
